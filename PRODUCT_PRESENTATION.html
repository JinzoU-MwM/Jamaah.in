8 Keputusan Desain Utama

Arsitektur yang membuat Jamaah.in cepat, andal, dan mudah dipelihara.

ü§ñ Gemini Vision ‚Üí JSON

Ekstraksi image-to-JSON langsung. Tanpa langkah OCR teks perantara. Akurasi jauh lebih tinggi untuk dokumen Indonesia (alamat tulis tangan, KTP latar batik).

üîÄ Fuzzy Merge Otomatis

Auto-merge KTP + Paspor + Visa untuk orang yang sama menggunakan SequenceMatcher (‚â•80% kecocokan nama). Eliminasi baris duplikat.

‚ö° Optimistic UI

Drag-and-drop dan hapus kamar langsung update UI, rollback otomatis jika API gagal. Menghilangkan latency ke Supabase.

üíæ Cache Frontend

Cache in-memory dengan TTL (subscription: 60 dtk, grup: 30 dtk). Auto-invalidate saat mutasi. Mencegah API call berlebihan.

üîí Optimistic Locking

Kolom version di Group mencegah konflik edit bersamaan. Response 409 Conflict saat version tidak cocok.

üîî Alert On-the-fly

Notifikasi dihasilkan dinamis dari data live ‚Äî tanpa tabel notifikasi tersimpan. Scan expired paspor, langganan, dan kelengkapan data.
